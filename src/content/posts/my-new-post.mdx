# 完全利用antigravity写一个个人博客·
好的，这是我用antigravity写的一个个人博客网站，初次搭建

# 使用的技术栈
- Next.js 14
- Tailwind CSS
- MDX
- Giscus

# 搭建过程

## 1. 创建项目

```bash
npx create-next-app@latest my-blog
```

## 2. 安装依赖

```bash
npm install next-mdx-remote rehype-pretty-code rehype-slug rehype-autolink-headings giscus
```

## 3. 配置

```typescript
// tailwind.config.ts
plugins: [
  require('@tailwindcss/typography'),
],
```

## 4. 编写内容

```markdown
---
title: 'Hello World: My First Post'
date: '2024-02-08'
description: 'This is a sample post to test the MDX setup.'
tags: ['nextjs', 'mdx', 'blog']
---

# Hello World

Welcome to my new blog! This post is written in **MDX**, which implies I can use React components inside markdown.
